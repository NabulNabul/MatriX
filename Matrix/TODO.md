# Matrix - Network Analyzer 개발 일지 및 To-Do List

## 📝 작업 완료 내용 요약

**1. 스미스 차트 (Smith Chart) 렌더링 고도화**
*   **정밀한 그리드 구현**: 표준 스미스 차트를 참고하여 임피던스/어드미턴스 원, 리액턴스/서셉턴스 호를 정교하게 렌더링.
*   **텍스트 레이블 표시**: 주요 저항(R) 및 리액턴스(X) 축에 수치 레이블을 추가하여 시각적 가독성 향상.
*   **VSWR 원 (Circle)**: 지정한 기준 VSWR(예: 2.0)에 해당하는 붉은 점선 원을 차트 위에 표시.

**2. 사용자 인터랙션 및 UX 강화**
*   **마우스 줌 & 패닝**: 마우스 휠을 이용한 부드러운 확대/축소(Zoom) 및 마우스 드래그를 통한 이동(Pan) 기능 구현 (더블클릭 시 뷰 초기화).
*   **실시간 좌표 정보**: 마우스 커서를 차트 위에 올리면 해당 지점의 임피던스(Z)와 반사계수(Γ) 실시간 표시.
*   **자동 매칭 추천 (클릭 매칭)**: 차트 클릭 시 목표 지점으로 이동하기 위해 필요한 단일 소자(L/C) 값을 역산하여 제안.

**3. 매칭 회로 설계 및 시뮬레이션**
*   **매칭 소자 UI**: 직렬/병렬 인덕터(L) 및 커패시터(C) 추가 버튼에 SVG 회로도 기호 적용.
*   **실시간 S-파라미터 연산**: 매칭 소자 변경 시 연쇄(Cascade) S-파라미터 실시간 재계산.
*   **매칭 경로 (Matching Path)**: 각 소자를 거칠 때마다 임피던스가 이동하는 경로를 녹색 점선으로 렌더링.
*   **주파수 선택 필터링**: 특정 주파수만을 선택하여 집중적으로 매칭 상태 확인 가능.

**4. 데이터 입출력 및 저장**
*   **Touchstone 파일 로드**: 계측기 측정 데이터(`.s1p`)를 불러와 주파수, MA/DB/RI 형식을 파싱하여 차트에 렌더링.
*   **매칭 회로 저장/불러오기**: 구성한 매칭 회로를 JSON 파일로 저장(`Save`) 및 복원(`Load`).
*   **차트 이미지 내보내기**: 현재 스미스 차트를 흰색 배경의 고해상도 PNG 파일로 저장.

**5. 지표 분석 및 빌드 환경 안정화**
*   **주요 지표(Metrics) 패널**: 현재 매칭 상태의 **VSWR** 및 **Return Loss** 실시간 표시.
*   **안정화**: WPF/Avalonia 충돌 해결, ReactiveUI 스레딩 이슈(`UseReactiveUI`) 해결, 바인딩 및 Avalonia 11 최신 문법 적용.
